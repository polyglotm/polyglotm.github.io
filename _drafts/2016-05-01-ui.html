기존 webpack기반 seed에서 bootstrap#v4.0.0-alpha.2
를 적용 하려고 2주간 온갑 삽질끝에 seed 프로젝트를 아래로 변경하였습니다.

https://github.com/mgechev/angular2-seed


새로운 seed는 gulp기반이며 bower를 사용하려면 추가적으로 설정을 해줘야 하지만
bootstrap이 npm기반으로도 제공되어 전혀 무리없이 바로 진행합니다.

gulp, webpack, grunt 다 써봤지만 역시 gulp가 제일 생산성 향상에 좋은듯합니다...

내친김에 bootstrap 4도 사용해봅니다.

$ npm install jquery --save-dev
$ npm install bootstrap@4.0.0-alpha.2 --save-dev


bootstrap4는 tether를 이용해서 tootip을 띄운다며 에러를 토해내길래 같이 설치해줍니다.

$ npm install tether --save-dev

(bootstrap4 공식 페이지에서도 해당 에러가 뜹니다. 무시하셔도 됩니다)
