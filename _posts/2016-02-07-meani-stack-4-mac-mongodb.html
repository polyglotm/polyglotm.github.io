[MEANI-Stack] - #4 Mac MongoDB 설치







이번에 진행할 내용은 Mac mongoDB를 설치입니다.






구글에 mongodb install mac으로 검색하니  공식사이트의 guide가 바로 나오네요.
https://docs.mongodb.org/manual/tutorial/install-mongodb-on-os-x/



거두절미하고 설치하는 방법은 터미널에서 아래 명령어를 실행하면 됩니다.
brew install mongodb






설치가 완료되었으면 mongoDB에서 사용할 기본 directory를 아래 명령어로 생성해 줍니다.
sudo mkdir -p /data/db





잘 설치되었는지 터미널에서 아래 명령어로 확인해 봅니다.
mongod












네 멋진 에러가 저를 반겨주는군요... 
하지만 설치는 되긴 된거같네요. 버전 3.2.1이라고 뜹니다.

바로 stack over flow를 검색합니다.









그렇습니다. 
서비스 시작을 시켜줘야지만 응답하는 당연한걸 안해줬네요...


stack over flow에 나온대로 아래 명령어를 실행합니다.

$ brew services start mongodb








네! 드디어 잘됩니다!
