이번에 소개할 내용은 환경변수를 cmd파일로 만들어 손쉽게 설정 하는것 입니다.

저같은경우는 아래와 같이 작성 하였습니다.
텍스트 문서로 작성하시고 확장자를 .cmd로 바꾸면 됩니다.

파일을 두가지로 나는 이유는
setx로 설정한 후에 바로 PATH에 설정하는게 먹히지 않았기 때문입니다.
--------------------------------------------------------------
첫번째 파일 - AutoPath_1.cmd

setx MAVEN_HOME "C:\apache-maven-3.1.1" /M
setx JAVA_HOME "C:\Program Files\Java\jdk1.7.0_55" /M
setx classpath ".;" /M
--------------------------------------------------------------

--------------------------------------------------------------
두번째 파일 - AutoPath_2.cmd

echo %MAVEN_HOME%
echo %JAVA_HOME%
echo classpath

setx PATH "%MAVEN_HOME%\bin; %JAVA_HOME%\bin; %PATH%" /M

@echo off

SET /P P=아무키나 입력하시면 종료됩니다~~~
--------------------------------------------------------------

echo기능을 통해 AutoPath_1.cmd가 제대로 동작하였는지 확인후
PATH에 입력하는 방식입니다.
