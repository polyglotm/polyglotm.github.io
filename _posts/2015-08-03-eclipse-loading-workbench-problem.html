eclipse loading workbench problem 

/ 이클립스가 열리지 않을때





무한 로딩에 빠지는경우 (Eternal Loading Case)

delete the file WORKSPACE/.metadata/.plugins/org.eclipse.core.resources/.snap


-----------------------------------------------------------------------------------------------------------------------


열리지도 않는경우는 ( If that does not open Case)

delete ".\workspace.metadata.plugins\org.eclipse.e4.workbench" folder.
