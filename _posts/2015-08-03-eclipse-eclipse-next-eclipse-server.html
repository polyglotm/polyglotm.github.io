[eclipse] eclipse 서버설정문제(next버튼이 비활성화 되었을때) / eclipse server setting problem (next button diable)
서버를 delete후 재설정 하려고 할때 next버튼이 비활성화 될 경우가 있습니다.

이때 해결방법을 제시해 드립니다.

1.이클립스종료
2.워크스페이스 폴더로 이동
workspace/.metadata/.plugins/org.eclipse.core.runtim/.settings 까지 이동
3. .settings 폴더안에있는
 3-1 org.eclipse.wst.server.core.prefs 삭제
 3-2 org.eclipse.jst.server.tomcat.core.prefs 삭제
4. 이클립스 실행후 서버 추가.

eclipse server setting problem (next button diable)

When you attempt to delete a server reset after next buttons may be disabled. 
We show how to solve this case. 

1 end of Eclipse 
2 Navigate to the folder where the workspace. 
[workspace / .metadata / .plugins / org.eclipse.core.runtim / .settings] to move 
3. [. Settings] in the folder 
  [org.eclipse.wst.server.core.prefs] Delete 
  [org.eclipse.jst.server.tomcat.core.prefs] Delete 

4. Add the server running Eclipse.
